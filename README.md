# MrAlbertoCompany

Sistema POS nativo para restaurantes, orientado a Windows 8.1 x86, con arquitectura limpia, operación local y enfoque táctil.

## Stack base

- C#
- .NET Framework 4.8
- WinForms
- SQLite local
- Dapper
- Impresión por `PrintDocument` y salida RAW opcional

## Estructura

- `src/` código de aplicación
- `docs/` arquitectura, datos, flujos y roadmap
- `installer/` empaquetado para cliente final

## Estado actual

Este repositorio contiene el esqueleto inicial de la solución, documentación de arquitectura y la fase 1 de base técnica para arranque, configuración local y bootstrap de base de datos.
