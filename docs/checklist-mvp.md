# Checklist de aceptación MVP

- Instala y ejecuta en Windows 8.1 x86.
- Arranca sin navegador, PHP ni Electron.
- Permite configurar negocio y usuario administrador inicial.
- Permite login por rol.
- Permite administrar categorías, productos y usuarios.
- Permite vender, imprimir y reimprimir tickets.
- Permite mesa, para llevar, mostrador y plataforma digital.
- Permite inventario opcional por producto.
- Permite solicitar y aprobar cancelaciones.
- Permite abrir y cerrar caja con trazabilidad.
- Permite respaldar y restaurar datos.
- Mantiene separación clara por capas.
